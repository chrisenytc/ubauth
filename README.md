## Getting Started

1º Clone ubauth repo

```bash
$ git clone git@github.com:chrisenytc/ubauth.git
```

2º Enter in ubauth directory
```bash
$ cd ubauth
```

3º Install dependencies

```bash
$ npm install
```

4º Run app

```bash
$ npm start
```

5º Test the app

```bash
$ npm test
```

6º Making releases

```bash
$ npm run release -- --sign identity_here
```
